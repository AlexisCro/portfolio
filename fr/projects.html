Converter

Converter est une petite application web principalement front faisant appel à 2 API afin de gérer les données.

L'application permet de convertir une devise en une autre, et offre la possibilité de visualiser sur un an l'évolution de devises en fonction d'une autre.

Les technologies utilisées :

  • HTML
  • CSS / Bootstrap
  • Javascript
  • API

Réalisée dans le cadre de ma reconversion, CasaVin est une web application réalisée en équipe dont je suis le responsable projet.

Travailler en équipe est une magnifique opportunité permettant l'approfondissement de ses connaissances via le partage de compétences et la mise en place et compréhension de workflow de développement.

Le but de cette application est de fournir un outil de gestion de cave à vin aux particuliers et de suivi de leurs vins (suivi de dégustations, suivi de stock, emplacement de la bouteille au sein de leur cave, ...). Une seconde version de cette application impliquera les professionnels et leur permettra de gérer leur cave ainsi que leur facturation. C'est à ce moment que l'application prendra tout son potentiel. Nous souhaitons fournir un outil de mise en relation entre les particuliers et les professionnels du vin de préférences locaux aux particuliers.

Ainsi nous souhaitons fournir un outil apportant une aide à la gestion de la cave des particuliers mais aussi un outil permettant la consommation locale ou la relation du caviste local avec le particulier. Le caviste pourra alors être force de proposition et d'aide au particulier.

Méthodologie SCRUM / Agile.

Les technologies utilisées :

  • HTML
  • CSS / Bootstrap
  • Javascript / Stimulus / Hotwire
  • Ruby on Rails
  • PostgreSQL
